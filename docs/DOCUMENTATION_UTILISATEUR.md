## Première connexion

### Inscription publique

1. Cliquez sur **"S'inscrire"** dans le menu
2. Remplissez le formulaire :
   - Numéro de téléphone (obligatoire)
   - Nom et prénoms
   - Email (optionnel)
   - Rôle souhaité (Scout, Parent, Consultant)
   - Mot de passe (minimum 6 caractères)
   - Acceptez le consentement RGPD
3. Soumettez le formulaire
4. **Important** : Votre compte sera en attente de validation par un administrateur
5. Une fois validé, vous recevrez une notification et pourrez vous connecter

### Connexion

1. Allez sur la page de connexion
2. Entrez votre numéro de téléphone et mot de passe
3. Si vous avez activé l'authentification à deux facteurs (MFA), entrez le code à 6 chiffres

### Configuration de l'authentification à deux facteurs (MFA)

1. Connectez-vous à votre compte
2. Allez dans **Mon profil** > **Sécurité**
3. Cliquez sur **Activer MFA**
4. Scannez le QR code avec une application d'authentification (Google Authenticator, Authy, etc.)
5. Entrez le code de vérification pour confirmer

## FAQ

### Comment réinitialiser mon mot de passe ?

Contactez un administrateur via le système de tickets ou par email.

### Mon compte est en attente de validation, que faire ?

C'est normal après l'inscription. Un administrateur validera votre compte sous 24-48h. Vous recevrez une notification une fois validé.

### Comment lier mon compte à mon profil scout ?

Contactez un administrateur qui pourra lier votre compte utilisateur à votre profil scout existant.

### Comment activer l'authentification à deux facteurs ?

1. Connectez-vous
2. Allez dans **Mon profil** > **Sécurité**
3. Cliquez sur **Activer MFA**
4. Suivez les instructions

### Comment faire une demande RGPD ?

1. Connectez-vous
2. Allez dans **Mon compte** > **Mes données**
3. Choisissez le type de demande
4. Remplissez le formulaire
5. Suivez l'état de votre demande

### Comment vérifier le statut ASCCI de mon enfant ?

Les parents peuvent voir le statut ASCCI dans les détails de leur enfant. Pour une vérification manuelle, contactez un gestionnaire.

### Comment contacter le support ?

Utilisez le système de tickets intégré dans l'application ou contactez directement les administrateurs.
